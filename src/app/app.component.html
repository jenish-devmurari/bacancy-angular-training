2 Why do we use ~ and ^ in the package.json file?

Both the symbols are use for version range to update the dependencies while install or update packages

For ~ symbol --- latest patch updated version is allow for-example ~1.2.3 is allow to update version less then 1.3.0 not (1.3.0)

For ^ symbol --- latest minor updated version is allow for-example ~1.2.3 is allow to update version less then 2.0.0 not (2.0.0)
